QMCPACK

Description

QMCPACK is the name of a Fortran 95 library for doing Quasi-Monte Carlo related computations. The main focus of the library is at:

Portability
The library is `portable' in the sense that the source code compiles under a variaty of compilers and adheres to the ISO/IEC 1539-1:1997 Fortran 95 standard. QMCPACK is being tested on a regular basis to compile with the following compilers:
  • G95
  • GNU Fortran 95
  • NAGWare f95 Compiler
  • Fortran Company/NAG F compiler
  • Intel(R) Fortran Compiler 9.0

Although this has not been tested, any other standard-conforming compiler should be able to compile the code to something executable. Feel free to send us your successfull attempts.

As Linux is currently the main development platform for QMCPACK, only Makefiles for Linux are included to build the executables. If you however succeed in compiling QMCPACK on Windows, please let us know how you did it and send us the necessary files.

Ease of use
Effort has been put in trying to keep the library as easy to use as possibly by implementing the algorithms with a clean, easy-to-use and intuitive interface.
Correctness
Several testmodules making up a kind of testsuite have been added so it is easy to check if certain changes break the code or not.
Speed?
In the beginning the main focus was of course on the correctness of the code. In the meanwhile, I observed however that some parts of it already have quite good timing-results. During the development, I therefore now not only stress on correctness, but also try to keep speed-issues in mind. Speed is however currently still not my primary focus.

Status

These things are currently as good as supported in the preliminary version of QMCPACK:

  • Generation of several low-discrepancy sequences, such as
    • Several radical inverse functions.
    • Halton sequences and several scramblings of it.
    • Sobol sequences.
    • Faure sequences (graycode or non-graycode ordening)
  • Randomizing pointsets with the Cranley-Patterson randomization technique.
  • Calculation of L2 discrepancy.
  • Test-integrals from Genz's testsuite and several other well known and often used testfunctions.
  • Several periodizing transformations as proposed by:
    • Sidi
    • Laurie
    • Sloan & Joe

Obtaining QMCPACK

As the software is still in it's early stages, there is no official release of QMCPACK yet. However, people who are interested can always mail me and ask for a copy.
